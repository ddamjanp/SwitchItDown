# Switch It Down

Switch It Down is a 2-player local fighting game built with Python and Pygame. The game features real-time combat where players can move, jump, block, and perform different attacks such as punches, kicks, and combos. 

Each player can switch between multiple forms during gameplay, including a ghost form focused on mobility and special attacks, and a stronger combat form with enhanced abilities. The game also includes environmental hazards and a health/meter system that affects how abilities are used, adding an extra layer of strategy.
## Technologies

- Python
- Pygame

## How to Run

1. Install dependencies:
pip install pygame

2. Run game:
python main.py
